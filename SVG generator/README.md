# Sharp Eleven SVG renderer experiments

This folder is a sandbox for evaluating a future full-SVG chart renderer.

## Font visualizer

Open `font-visualizer.html` to compare fonts by musical role:

- root letter
- accidentals
- quality text
- superscript figures
- music symbols
- slash bass

The visualizer has two modes:

- `Texte live`: uses the font files directly through `@font-face`.
- `SVG atlas`: composes chords from extracted glyph paths generated by `generate_atlas.py`.

## Generate glyph atlases

From this folder:

```powershell
python .\generate_atlas.py
```

Useful options:

```powershell
python .\generate_atlas.py --font-id app-noto-latin
python .\generate_atlas.py --chars "ø°△♭♯"
python .\generate_atlas.py --chars-file ".\accords test.txt"
```

The script writes both JSON and browser-loadable JS files under `atlases/`.
The JS files make `font-visualizer.html` usable even when opened directly from disk.
